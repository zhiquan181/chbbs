# chbbs
本项目是基于SSH的社区论坛系统的设计与实现

<br/>

运行环境与工具：<br/>
Win8.1系统<br/>
MySQL5数据库<br/>
JDK版本jdk1.8.0_91<br/>
Apache Tomcat7服务器<br/>
MyEclipse 10编译器<br/>
Navicat for Mysql可视化软件等。<br/>

<br/>

技术：<br/>
采用Java EE的SSH框架，基本的CURD操作和传参数,前端使用Jquery<br/>

<br/>

描述：<br/>
这个是J2EE的毕设，主要功能是发帖与评论，以及后台系统的管理。
<br/>

感言：<br/>
这个项目做得一般，不过不打算翻新，只是记录一些学习的东西。<br/>
